The Application

This web application as a first in Angular 4 for John is designed to show case its basic features such as:

  • Animation
  • Reactive Forms
  • Code/Component Resusability
  • Shortened Development LifeCycle - Ease of Deployment

Its content is comprised of two basic tenants, firstly some information and images about Dublin, Ireland for the prospective visitor. The application also has Angular 4 tips/links, which are extremely useful if you are new to Angular like John or moving from Angular 1/2 to Angular 4. The Application front end does not have any database build to house data generated from the online contact form but a backend maybe implemented into the future. For this version 1.0 application, the front end will not have persistent storage capability. There is however good responsiveness built into the front end which is tested on Chrome, Firefox and Safari passing tests run on the latest versions of these browsers. Also, semantic HTML5 language has been used to make the application more acceessible for the visually impaired. The responsiveness of the pages is indicative of the site's design focus, which is:

  • Brochure Styled Application Front End Design
  • Single Page Application Architecture Utilising Angular 4's Asynchronous Features
  • Use of Responsive Design Techniques
  • Use of Modern Languages like HTML5, CSS3 and Typescript

The user experience is designed to be pleasant, welcoming and coordinated for the most user friendly delivery of information per the design objectives for the user.
